My favorite Learning Resources!

Duolingo

A website that makes learning languages feel more like a game than studying! It is also available as an app.

SpanishDict

A spanish dictionary website that has great sentence examples as well as flashcard learning resources.

Hello Talk

Hello Talk is an app that allows you to talk to native speakers! Great for practicing what you learn and making friends all over the world.

Study Spanish

This website is an amazing resource for all things grammar.
